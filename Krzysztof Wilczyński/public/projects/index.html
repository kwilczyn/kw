  • Estetyczna Tychy

This is a static web application built from scratch using Vanilla JS and SCSS. Why Vanilla JS? Because I wanted to see if I could write this type of application without using modern frameworks like Vue.js or React.

The project is covered with unit tests written using the Vitest library.

Vite is responsible for the application build process.

For deployment purposes, a workflow was prepared in Github Actions

E2E tests (written using Playwright) are performed on the staging environment for various browser sizes, including mobile devices (the layout used utilizes media queries).
